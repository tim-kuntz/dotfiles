# README

## Dependencies

Installation relies on [GNU Stow](https://www.gnu.org/software/stow/) ([doc](https://www.gnu.org/software/stow/manual/stow.html)).

On Mac
```bash
brew install stow
```

## Installation
```bash
git clone https://github.com/tim-kuntz/dotfiles.git
cd dotfiles
stow <package>
```
